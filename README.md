# Echo 💬

**Echo** — корпоративный мессенджер для командной работы с удобным интерфейсом, системой аккаунтов и возможностью масштабирования под нужды компании.

Проект состоит из frontend и backend частей и предназначен для внутреннего общения сотрудников.

---

## 🚀 Основные возможности

- 💬 Личные и рабочие чаты  
- 🏢 Рабочие пространства (Whiteboard)  
- 📊 Панель управления (Dashboard)  
- 🔐 Система аккаунтов и авторизации  
- 🎨 Современный UI с кастомными стилями  
- 📁 Поддержка медиа и шрифтов  
- ⚡ Быстрый интерфейс на React
